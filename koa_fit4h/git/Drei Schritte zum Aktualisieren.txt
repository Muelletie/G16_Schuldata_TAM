Drei Schritte zum Aktualisieren

1. Sicherstellen, dass Sie in der Basis des verwalteten 	Verzeichnisses sind (/home/pan/Dokumente/KuA_Fit4h)

2. git add . -A
	# Der  .  steht für das aktuelle Verzeichnis
	# -A steht für alle geänderten, hinzugefügten, gelöschten Datein

3. git commit -am 'Was passiert ist'
	#Bestätigung der Änderung lokal

4. git push origin master
	#Hochladen der neuen Version auf den Server
